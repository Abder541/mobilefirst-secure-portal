# Recréation Mobile-First — Secure Portal (240 pts)

## But
Mini-site inspiré d’un portail de connexion pour démontrer une approche **Mobile-First**
et une réflexion **UCD + cybersécurité** (prévention des erreurs humaines).

## Pages
- `index.html` : présentation + principes
- `login.html` : connexion
- `reset.html` : réinitialisation (anti-énumération)
- `dashboard.html` : exemple post-connexion

## Tech
- HTML / CSS (Mobile-First)
- JavaScript optionnel (`Afficher/Masquer` le mot de passe)

## Déploiement
GitHub Pages → Branch `main` → Folder `/ (root)`.
